# Проект: Место

Базовый проект по JS без использования бэкенда.

## Возможности

- Валидация форм
- Редактировать профиль (имя, род деятельности)
- Добавлять/удвлять новые карточки
- Просмотр картинки в масштабе, через поп-ап

### Технологии

- Javascript
- Full Responsive
- Desktop first
- Grid
- Flex
- Position
- BEM
- Nested Bem
